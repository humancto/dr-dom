Get Started in 2 Minutes

No signup, no API keys, just install and go

1

Clone the Repository

Get the code from GitHub

git clone https://github.com/humancto/dr-dom.git
2

Open Chrome Extensions

Navigate to Chrome's extension settings

chrome://extensions/
3

Load Extension

Enable "Developer mode" → Click "Load unpacked" → Select the dist/chrome folder
